TechPicks Chatbot

TechPicks Chatbot은 전자기기에 대한 정보를 쉽고 빠르게 알려주는 카카오톡 기반 AI 챗봇입니다.

본 프로젝트는 Beyond Class 1 프로그램을 통해 '비요뜨팀'이 개발한 챗봇 서비스로, 사용자들이 카카오톡 채팅을 통해 다양한 전자기기 정보를 자연스럽게 탐색할 수 있도록 설계되었습니다.

AWS 인프라 위에 Flask 기반의 AI 서버를 구축하고, 카카오톡 채팅 API와 연동하여 실제 대화형 서비스를 구현하였습니다. 챗봇은 사용자의 질문을 분석하여 노트북, 스마트폰, 주변기기 등 다양한 전자기기에 대한 스펙, 추천 정보 등을 제공합니다.

팀은 챗봇의 응답 정확도를 높이기 위해 대화 흐름 설계와 자연어 처리 개선에 집중하였으며, 사용자 피드백을 반영해 지속적으로 모델을 개선해 나갔습니다.

TechPicks Chatbot은 단순한 검색을 넘어, 사용자와의 대화 속에서 유용한 정보를 제안하고 전자기기 선택을 도와주는 ‘디지털 쇼핑 가이드’ 역할을 지향합니다.

이번 프로젝트를 통해 팀원들은 실제 챗봇 서비스 개발 과정과 서버 인프라 운영, API 통신 흐름에 대해 깊이 있는 실무 경험을 쌓을 수 있었습니다.
